### Bruto force

```java
public int[] twoSum(int[] nums, int target) {
    for (int i = 1; i < nums.length; i++) {
        int j = 0;
        while (j < i) {
            int sum;
            if ((sum = nums[i] + nums[j]) == target) {
                return new int[]{i, j};
            }
            j++;
        }
    }
    return new int[]{};
}
```
